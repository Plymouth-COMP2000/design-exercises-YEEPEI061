# Project Title
An Android-based restaurant management application developed using Java and XML, enabling staff to manage menus and reservations and allowing guests to browse menus and make reservations. The system integrates a RESTful API and SQLite to ensure efficient data handling and smooth user experience.

## Technologies Used
- Android (Java)
- SQLite
- Volley (HTTP networking)
- JSON
- REST API

## Third-Party Resources Used
The following third-party resources were used during the development of this project:

### Libraries & Frameworks
- **Android Volley** – Used for handling HTTP requests and API communication.
- **Gson** – Used for JSON parsing.
- **SQLite** – Used for local data storage.

### APIs
- **Coursework REST API** – Provided by the module for user authentication and data management.

### Tools
- **Android Studio** – Development environment.
- **GitHub** – Version control.
